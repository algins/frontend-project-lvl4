### Requirements:
* Git
* Make
* Node.js ^17
* Heroku CLI

### Setup:
```sh
$ git clone https://github.com/algins/frontend-project-lvl4.git
$ cd frontend-project-lvl4
$ make install
```

### Run:
```sh
$ make start # Open http://localhost:5000
```
